=== Citation ===
Tested up to: 6.0.1
Requires PHP: 7.3.5

Add citation box in Post Admin area.


== Description ==
Add citation box in admin area to complete the citations in each post.
Is possilbe to call the content of the citation box whit the shortcode [citation].

Shortcode options:

If you use only [citation] you will get the citation box of the current post.

If you use [citation post-id=\"XX\"] you will get the citation box of the post id XX.

== Installation ==
Go to:

1. Plugins > Add New > Upload Plugin

2. Drag and drop the ZIP file.

3. Install and Activate

Is ready to use, no aditional set-up was requered
